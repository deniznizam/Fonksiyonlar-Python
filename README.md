# Fonksiyonlar-Python
Bu proje, python dilinde temel fonksiyon kullanımını göstermek amacıyla hazırlanmıştır.  İçerik: • Pi ve yarıçap bilgisi ile dairenin alanını hesaplayan fonksiyon • Döngü kullanarak faktöriyel hesaplama fonksiyonu • Doğum yılına göre kişinin yaşını hesaplayan fonksiyon .Fonksiyon içinde fonksiyon kullanılarak emeklilik durumunu hesaplayan fonksiyon
